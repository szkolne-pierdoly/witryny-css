Co znaczy skrót ID? Skrót z języka angielskiego: Identificator - po poslku Identyfikator - oznacza przeważnie numer identyfikacyjny np. obiektu w bazie danych lub elementu na stronie HTML
Jaki jest cel stosowania ID? W HTML używamy ID aby oznaczyć dany obiekt (np. div lub button) daną nazwą aby później móc się odwołać do niego w CSS lub JS
Jakie jest znaczenie znaku # znak '#' jest to początek selektora do id w CSS - stworzono go aby css rozpoznawał czy odwołujemy się do id lub klasy
Jak następuje wywołanie ID Aby nadać id danemu obiektowi wystarczy w tagu HTML dodać 'ID="nazwa"' - gdzie nazwa stanowi nazwę id jakie chcemy nadać obiektowi
Uwagi na temat stosowania ID
  • Id może wystąpić tylko raz w całym dokumencie html, zaś można użyć tego same ID w dwóch plikach np. strona główna i podstrona
  • nie można używać w nazwie ID spacji, ponieważ spowoduje to nadanie dwóch id do jednego obiektu np. id="dwa slowa" spowoduje nadanie jednemu obiektowi id 'dwa' oraz id 'slowa'
  • Id musi być unikatowe ponieważ przy bardziej zaawansowanych stronach lub aplikacjach internetowych JavaScript nie byłby w stanie działać poprawnie

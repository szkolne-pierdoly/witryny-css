  1. Cel stosowania szablonów HTML 4.01 Transitional

    Szablon HTML 4.01 Transitional jest stosowany w celu zapewnienia kompatybilności ze starszymi przeglądarkami.

  2. Opisz 'strict frame', 'transitional', i 'frame set'

    HTML 4.01 to czwarta wersja języka HTML, która wprowadziła trzy rodzaje DTD (Document Type Definitions) - Strict, Transitional i Frame Set. Każdy z nich ma swoje cechy i zastosowanie.

    1. Strict

      • Jest najbardziej rygorystycznym typem DTD w HTML 4.01.
      • Nie pozwala na używanie elementów i atrybutów, które są uważane za przestarzałe lub niezgodne z normami HTML.
      • Ten typ DTD jest zwykle wykorzystywany do tworzenia stron internetowych zgodnych z normami W3C.

    2. Transitional

      • Jest najbardziej elastycznym typem DTD w HTML 4.01.
      • Pozwala na używanie elementów i atrybutów, które są uważane za przestarzałe lub niezgodne z normami HTML.
      • Ten typ DTD jest zwykle wykorzystywany do tworzenia stron internetowych, które mają być kompatybilne z przeglądarkami internetowymi starszych wersji.

    3. Frame Set

      • Jest to typ DTD, który pozwala na tworzenie stron internetowych z ramkami (ang. frames).
      • Elementy i atrybuty, które są uważane za przestarzałe lub niezgodne z normami HTML, są dozwolone w tym typie DTD.
      • Ten typ DTD jest zwykle wykorzystywany do tworzenia stron internetowych, które wykorzystują ramki do podziału treści na różne sekcje.

  3. Opisy i przykłady html 4.01 - tabela

    tytuł Przykład Opis
    Szablon html 4.01 transitional z tabelą Ten szablon zawiera tabelę HTML, która jest poprawnie sformatowana zgodnie ze standardami HTML 4.01 Transitional. W tym szablonie zastosowano elementy HTML takie jak "table", "tr", "th" i "td", które są dozwolone w HTML 4.01 Transitional.
    Szablon HTML 4.01 Transitional z linkami Ten szablon zawiera listę linków HTML, która jest poprawnie sformatowana zgodnie ze standardami HTML 4.01 Transitional. W tym szablonie zastosowano elementy HTML takie jak "h1", "ul" i "li", które są dozwolone w HTML 4.01 Transitional, oraz element "a", który służy do tworzenia linków.
    Szablon HTML 4.01 Transitional z obrazkiem Ten szablon zawiera obrazek HTML, który jest poprawnie sformatowany zgodnie ze standardami HTML 4.01 Transitional. W tym szablonie zastosowano elementy HTML takie jak "h1" i "p", które są dozwolone w HTML 4.01 Transitional, oraz element "img", który służy do wyświetlania obrazków na stronie internetowej

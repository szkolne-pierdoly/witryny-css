Wykorzystanie znacznika LINK

Umieszczamy w tagu < head >

< link rel="stylesheet" type="text/css" href="plik.css" />

Wykorzystanie znacznika STYLE

Umieszczamy w tagu < head >

< style >@import url('plik.css');< /style >

Wykorzystanie znacznika SCRIPT

Umieszczamy w tagu < head >

< script type="text/javascript" src="plik.css">< /script >

Wykorzystanie odwołania do pliku css w atrybucie CLASS

Umieszczamy w dowolnym tagu

< body class="plik-css"> ... < /body>

Wykorzystanie odwołania do pliku css w atrybucie STYLE

Umieszczamy w dowolnym tagu

< body style="background: url(plik.css);" > ... < /body>

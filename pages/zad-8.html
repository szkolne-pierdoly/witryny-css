gdzie jest wstawiany?

Arkusz stylów jest wstawiany w sekcji dokumentu HTML.

składnia teoretyczna

Składnia teoretyczna arkusza stylów wygląda następująco:
< style type="text/css">
selektor:pseudocecha { atrybut: wartość }
np.
przycisk1:hover {
color: blue
}
< /style>

jakie mogą być selektory

Selektory mogą być klasami, id, wyrażeniami regularnymi, selektorami elementu lub atrybutem.

jakie mogą być cechy i wartości

Cechy i wartości obejmują kolory, czcionki, marginesy, pozycjonowanie, szerokość, wysokość, efekty wizualne i wiele innych.
